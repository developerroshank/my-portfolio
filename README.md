## 🔧 **Setup & Installation**
1. **Clone the Repository**  
   ```sh
   git clone https://github.com/developerroshank/my-portfolio
   cd my-portfolio
   ```
2. **Install Dependencies**  
   ```sh
   npm install
   ```
3. **Run the Development Server**  
   ```sh
   npm run dev
   ```
4. **Open in Browser**  
   Go to `http://localhost:3000`
